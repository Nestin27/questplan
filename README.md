# QuestPlan

> **Планируем жизнь к лучшему** — планировщик задач, блокнот и платёжный трекер в одном приложении.

---

## Возможности

### 📅 Планировщик
- Недельный календарь в виде плиток-карточек со скруглёнными углами
- Числа месяца в каждом дне, подсветка «сегодня»
- Drag & Drop — перетяните задачу на нужный день
- Выдвигающаяся боковая панель задач (на мобиле — шторка снизу)
- Задачи-квесты: подзадачи с прогресс-баром + заметки к каждой задаче
- Быстрое добавление задачи прямо в ячейке дня

### 📓 Блокнот
- Карточки с заголовком и текстом
- Блоки кода с подсветкой синтаксиса и выбором языка (JS, Python, SQL, Bash и др.)
- Кнопка копировать с подтверждением «Скопировано!»
- Прикрепление файлов: изображения показываются как превью, документы — как плитки со скачиванием
- Лайтбокс для просмотра фото прямо на странице

### 💳 Платёжный планировщик
- Таблица платежей × 12 месяцев
- Отметка оплаты галочкой для каждого месяца
- Умные ячейки: ✅ оплачено / ❗ просрочено / 🕐 сегодня день платежа / 📅 ещё не наступило
- Поддержка нескольких валют (₽ $ € £ ₴ ₸) — итоги считаются отдельно по каждой валюте
- День платежа — цветной бейдж в таблице
- Итого по месяцам и за год в футере таблицы

### 🌍 Мультиязычность
Переключение языка в нижней панели:
- 🇷🇺 Русский (по умолчанию)
- 🇬🇧 English
- 🇩🇪 Deutsch
- 🇨🇳 中文

### 🎨 Темы
- ☀︎ Светлая
- ☾ Тёмная
- ⊙ Авто (системная)

### 📱 Адаптивность
- Полноценная работа на мобильных устройствах
- На телефоне — 2 колонки в календаре
- Панель задач выезжает снизу как шторка
- Все элементы управления адаптированы под касание

## Стек

| Слой | Технология |
|------|-----------|
| Frontend | React 18 + Vite |
| Backend  | Node.js + Express |
| База данных | SQLite (better-sqlite3) |
| Файлы | Хранятся на сервере в `/data/uploads` |
| Иконки | Векторные SVG (без эмодзи) |

## Быстрый старт (локально)

```bash
# 1. Распаковать и установить зависимости
tar -xzf questplan.tar.gz
cd questplan
npm install

# 2. Запустить бэкенд (Терминал 1)
npm run dev:server
# → API: http://localhost:4000

# 3. Запустить фронтенд (Терминал 2)
npm run dev:client
# → Приложение: http://localhost:3000
```

**Открыть с телефона или другого устройства в локальной сети:**
```
http://<IP вашего компьютера>:3000
```
Узнать IP: `ipconfig` (Windows) или `ip a` (Linux/Mac)

---

## Docker

```bash
# Собрать и запустить
docker compose up -d --build

# Приложение: http://localhost:8080
# В локальной сети: http://<IP>:8080

# Логи
docker compose logs -f

# Остановить
docker compose down
```

Данные (SQLite + файлы) хранятся в Docker volume `questplan_data` — не теряются при перезапуске.

## API

| Метод | Маршрут | Описание |
|-------|---------|----------|
| GET | `/api/tasks` | Все задачи с подзадачами и заметками |
| POST | `/api/tasks` | Создать задачу |
| PATCH | `/api/tasks/:id` | Обновить задачу (название, статус, день) |
| DELETE | `/api/tasks/:id` | Удалить задачу |
| POST | `/api/tasks/:id/subtasks` | Добавить подзадачу |
| PATCH | `/api/tasks/:id/subtasks/:sid` | Отметить подзадачу |
| DELETE | `/api/tasks/:id/subtasks/:sid` | Удалить подзадачу |
| POST | `/api/tasks/:id/notes` | Добавить заметку к задаче |
| DELETE | `/api/tasks/:id/notes/:nid` | Удалить заметку задачи |
| GET | `/api/notes` | Все заметки блокнота |
| POST | `/api/notes` | Создать заметку |
| PATCH | `/api/notes/:id` | Обновить заголовок/текст |
| DELETE | `/api/notes/:id` | Удалить заметку |
| POST | `/api/notes/:id/blocks` | Добавить блок (текст/код) |
| PATCH | `/api/notes/:id/blocks/:bid` | Обновить блок |
| DELETE | `/api/notes/:id/blocks/:bid` | Удалить блок |
| POST | `/api/notes/:id/blocks/file` | Загрузить файл (multipart) |
| GET | `/uploads/:filename` | Скачать/просмотреть файл |
| GET | `/api/payments` | Все платежи с историей оплат |
| POST | `/api/payments` | Создать платёж |
| PATCH | `/api/payments/:id` | Обновить платёж |
| DELETE | `/api/payments/:id` | Удалить платёж |
| POST | `/api/payments/:id/check` | Отметить/снять оплату за месяц |
